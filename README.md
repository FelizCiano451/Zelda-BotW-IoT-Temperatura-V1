Passo 01: Instalar as bibliotecas do Python com o comando:

pip install -r requirements.txt

Passo 02: Garantir o suporte a vídeo

No Windows: No prompt de comando basta rodar:

winget install Gyan.FFmpeg

No Linux: Basta abrir o terminal da máquina e rodar:

sudo apt update && sudo apt install ffmpeg -y
